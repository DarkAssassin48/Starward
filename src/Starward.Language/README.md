# Starward.Language

Этот проект является единственным местом хранения локализации Starward.

## Правила

- Все пользовательские строки должны находиться в `Lang.resx` и соответствующих `Lang.<culture>.resx`.
- Форматтеры, зависящие от языка или региональных настроек, должны находиться в этом проекте.
- В проектах `Starward`, `Starward.Core` и других проектах приложения нельзя добавлять массивы переводов, проверки языка и жёстко заданные локализованные строки.
- Код интерфейса должен получать текст через `Lang` или публичные форматтеры из пространства имён `Starward.Language`.
- Английские значения в `Lang.resx` используются как резервные для отсутствующих переводов.

`LocalizedTimeFormatter.cs` содержит общую локализацию длительности и подписей игрового времени. Форматы единиц времени читаются из ресурсов `Lang*.resx`.

Эта временная строка используется только для запуска проверочной сборки и не предназначена для слияния.
